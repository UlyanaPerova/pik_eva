# PIK EVA — Оркестрант недвижимости

Система сбора и анализа данных по кладовкам и квартирам от застройщиков. Парсит сайты, считает EVA-баллы, выгружает в Excel.

## Быстрый старт (для новичков)

### Шаг 1. Скачай проект

Нажми зелёную кнопку **Code** вверху этой страницы, затем **Download ZIP**.

Распакуй архив в любое удобное место (например, на Рабочий стол).

> Если у тебя установлен Git — можно вместо этого открыть терминал и написать:
> ```
> git clone https://github.com/ТВОЙ_АККАУНТ/pik_eva.git
> ```

### Шаг 2. Установи Python

Если Python ещё не установлен:

1. Перейди на [python.org/downloads](https://python.org/downloads)
2. Скачай последнюю версию
3. При установке **обязательно** поставь галочку **"Add Python to PATH"**

### Шаг 3. Запусти установку

Открой папку с проектом и **дважды кликни** на файл:

- **Windows:** `setup.bat`
- **macOS:** `start.command`

Установка сделает всё сама: создаст окружение, скачает зависимости, установит браузер для парсинга. На рабочем столе появится ярлык **PIK EVA**.

Это нужно сделать **только один раз**.

### Шаг 4. Запусти программу

**Дважды кликни** на ярлык **PIK EVA** на рабочем столе (или на файл `start.bat`).

Откроется браузер с интерфейсом программы. Готово!

## Как пользоваться

Программа открывается в браузере. Слева — панель навигации с 5 вкладками:

### Пульс
Главная страница. Показывает состояние всех застройщиков: сколько кладовок и квартир в базе, когда последний раз обновлялись данные.

### Сбор
Здесь запускается парсинг. Выбери нужных застройщиков (галочки), нажми **Запустить сбор**. Застройщики парсятся параллельно. Логи выполнения отображаются в реальном времени. Здесь же кнопка автозапуска: она работает при соблюдении следующих условий — время после 11АМ, подключение к интернету и включенный компьютер (что очевидно).

<img width="1243" height="727" alt="image" src="https://github.com/user-attachments/assets/a0d5ec3d-886f-4424-ade9-7bf8384585d9" />

### Формулы
Редактор формул EVA. Три вкладки:
- **Первый этап** — пороги баллов (срок ввода, площадь, стоимость и т.д.)
- **Второй этап** — параметры Excel-формул
- **Алиасы** — маппинг названий ЖК и корпусов между застройщиками и ДОМ.РФ

Не забудь нажать **Сохранить формулы** после изменений. Кнопка **Сбросить** вернёт всё к исходным значениям.

<img width="1470" height="719" alt="image" src="https://github.com/user-attachments/assets/4932fce7-2010-4866-9805-41be763c6e5c" />

### Связи
Управление ссылками ДОМ.РФ. Здесь добавляются и удаляются объекты для парсинга с ДОМ.РФ. Для добавления нужен только **Object ID** (число из ссылки на объект) и название ЖК.

<img width="1450" height="746" alt="image" src="https://github.com/user-attachments/assets/3f7ffc52-10f6-4ad0-b50d-35cb60e8865d" />

### Обновление
- **Git Pull** — обновить программу до последней версии (нужен Git)
- **Собрать расчет EVA** — сгенерировать сводный файл `расчет_ева.xlsx`
- **Логи** — посмотреть историю выполнения

## Как обновить программу

Если установлен Git:
1. Открой программу
2. Перейди на вкладку **Обновление**
3. Нажми **Git Pull**

Если Git не установлен:
1. Скачай новый ZIP с GitHub (как в Шаге 1)
2. Замени файлы в папке проекта

## Структура проекта

```
pik_eva/
├── start.bat          ← Запуск (Windows)
├── start.command      ← Запуск (macOS)
├── setup.bat          ← Первоначальная установка
├── icon.ico           ← Иконка приложения
│
├── site/
│   ├── api.py         ← Сервер (FastAPI)
│   └── index.html     ← Веб-интерфейс
│
├── orchestrator.py    ← Ядро: запуск парсеров
├── config_manager.py  ← Управление конфигами
├── autostart.py       ← Автозапуск при старте Windows
│
├── configs/           ← Настройки (YAML)
├── parsers/           ← Парсеры сайтов
├── runners/           ← Скрипты запуска
├── core/              ← Расчёт EVA
├── exporters/         ← Выгрузка в Excel
├── data/              ← Базы данных (SQLite)
└── output/            ← Результаты (xlsx)
```

## Если что-то не работает

**Браузер не открылся:** подожди 10 секунд и открой вручную [http://localhost:8090](http://localhost:8090)

**Ошибка "Python не найден":** установи Python и обязательно отметь "Add Python to PATH" при установке

**Парсинг не запускается:** проверь подключение к интернету

**Кнопка Git Pull не работает:** установи Git с [git-scm.com](https://git-scm.com/download/win)
